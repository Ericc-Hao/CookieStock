Gin Rummy, With a Twist

Overview

Welcome to Gin Rummy, With a Twist! This project is a digital recreation of the classic two-player card game, Gin Rummy, with an exciting twist: itâ€™s played using a base-twelve (dozenal) number system. With a deck of 64 cards, it combines traditional gameplay with fresh strategies for a new challenge. We aim to provide a smooth, engaging experience with rich animations and intuitive user interactions.

Features

  • Dozenal Game Logic: Play using a base-twelve scoring system, offering a fresh take on Gin Rummy strategies.
  • Interactive Gameplay: Smooth animations for dealing and sorting cards, responsive interactions, and clear game state updates.
  • Multiplayer Support: Engage in matches with other players through online matchmaking or invite friends.
  • User Profiles & Rankings: Track your progress and rank on a global leaderboard.
  • Cross-Platform Compatibility: Responsive UI for desktop, mobile, and tablet.

Tech Stack

Frontend: Next.js, Redux, Shadcn/ui for a responsive, dynamic interface.

Backend: Spring Boot for API and game logic, PostgreSQL for database management.

Hosting: Deployed on Google Cloud Platform for scalability and availability.
